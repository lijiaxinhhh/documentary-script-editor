# Product Context

本项目是一个本地优先的纪录片文稿剪辑工作台。用户雇佣它完成的任务不是“看一个漂亮官网”，而是尽快把本地采访素材变成可搜索逐字稿、可拖拽故事版和可导出到剪辑软件的粗剪工程。

## Primary User

- 独立纪录片作者、剪辑师、小型制作团队。
- 经常处理采访、观察性素材、档案资料、B-roll 和长时间码。
- 需要减少上传素材、手动找时间码、纸剪辑和剪辑软件之间反复搬运的成本。

## Core Task

1. 导入本地视频或恢复本地项目。
2. 输入自己的模型 Key 或使用本机转写执行层。
3. 得到带发言人和时间码的逐字稿。
4. 通过文字播放视频、搜索原话、改发言人、高亮选段。
5. 把选段和便签组织成故事版。
6. 导出给 Final Cut Pro、Premiere Pro、DaVinci Resolve、剪映/CapCut 或本地项目文件。

## Product Boundaries

- 不做营销首页，第一屏就是工作台。
- 不假装云端转写已经完成；执行层不可用时要清楚显示阻塞状态。
- 不把本地视频对象地址写进可导出的项目文件；恢复项目后必须提示重新关联素材。
- 设计风格服务重复工作：密度高但有秩序，少装饰，强命令入口，保留真实编辑器的可扫描性。
